# @uniicy/ui-beyond

Compliance-as-Code design system for Uniicy iGaming platforms. Pre-built React components covering KYC, AML, risk, transactions, alerts, admin UIs.

## Install

Scoped to GitHub Packages. Configure `.npmrc` in your project:

```ini
@uniicy:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=${GITHUB_TOKEN}
```

Then install:

```sh
bun add @uniicy/ui-beyond
# or
pnpm add @uniicy/ui-beyond
# or
npm install @uniicy/ui-beyond
```

`GITHUB_TOKEN` must be a Personal Access Token with `read:packages` scope.

## Use

```tsx
import '@uniicy/ui-beyond/styles'
import { Button, Badge, ThemeProvider } from '@uniicy/ui-beyond'

export function App() {
  return (
    <ThemeProvider>
      <Button variant="primary">Verify Identity</Button>
      <Badge variant="approved" dot />
    </ThemeProvider>
  )
}
```
